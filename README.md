# Tu Web

Landing page para ofrecer servicios de diseño web a personas y pequeños negocios.

## Stack

HTML + CSS + JavaScript vanilla. **0 dependencias.**

## Secciones

| # | Sección |
|---|---------|
| 1 | **Header** — Nav con logo, 5 enlaces, menú hamburguesa |
| 2 | **Hero** — Titular, CTAs, mockup animado |
| 3 | **Diseños** — Carrusel con 6 estilos (auto-play, táctil, botones) |
| 4 | **Servicios** — 4 tarjetas con íconos SVG |
| 5 | **Precios** — 3 planes (Básico $499, Profesional $999, Premium $1,999 MXN) |
| 6 | **Contacto** — Formulario con validación |

## Los 6 estilos de diseño

Minimalista · Corporativo · Moderno · E-commerce · Portfolio Personal · Landing Page

Cada uno abre una vista detallada con mockup completo del sitio y paleta de colores.

## Uso

```bash
python3 -m http.server 8080
# Abrir http://localhost:8080
```

O simplemente abrir `index.html` en el navegador.
